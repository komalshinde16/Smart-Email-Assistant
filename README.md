The backend of the Smart Email Assistant is built using Spring Boot, following a layered architecture with Controller and Service components. It exposes a RESTful API (/api/email/generate) that accepts email content and tone as input and returns an AI-generated response. The application handles JSON requests, processes business logic in the service layer, and ensures smooth client-server communication with proper CORS configuration. API endpoints were thoroughly tested using Postman to validate request-response cycles, handle edge cases, and ensure reliability and performance of the system.
